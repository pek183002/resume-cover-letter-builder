# 🎯 AI-Powered ATS Resume & Cover Letter Suite

An intelligent, Python-based career tool that helps job seekers bypass Applicant Tracking Systems (ATS) by tailoring resumes and cover letters using Google's Gemini AI.

## ✨ Key Features
* **Resume Parsing:** Automatically extracts text from your existing PDF master resume.
* **ATS Match Scoring:** Get a 0-100% compatibility score against any job description.
* **Keyword Optimization:** Identifies missing industry keywords and integrates them naturally.
* **Tailored Cover Letters:** Generates persuasive letters that focus on cultural fit and specific job requirements.
* **ATS-Friendly PDF Export:** Generates "flat" PDF files designed specifically for seamless parsing by systems like Workday, Greenhouse, and Lever.

## 🛠️ Tech Stack
* **Language:** Python 3.9+
* **AI Engine:** Google Gemini 1.5 Flash
* **Web Framework:** [Streamlit](https://streamlit.io/)
* **PDF Processing:** PyPDF2 & FPDF2
